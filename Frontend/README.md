# Frontend - execucao com Vite

Este projeto foi configurado para rodar localmente apenas com o Vite.

## Requisitos

- Node.js 20+
- npm 10+

## Como rodar

```bash
cd Frontend
npm install
npm run dev
```

O app sera iniciado por padrao em `http://localhost:5173`.

## Scripts

- `npm run dev`: inicia o servidor de desenvolvimento Vite
- `npm run build`: gera build de producao
- `npm run preview`: faz preview local do build
- `npm run lint`: executa o ESLint
